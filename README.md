# FizzBuzz Function

**how to use?**

For console log:
  console.log(fizzBuzz(begin,end);
  
For return:
  return fizzBuzz(begin, end);

Example: 
console.log(fizzBuzz(13, 20))
  
  Example output:
  
1.  **11**
2.  **Fizz**
3.  **13**
4.  **14**
5.  **FizzBuzz**
6.  **16**
7.  **17**
8.  **Fizz**
9.  **19**
10. **Buzz**
